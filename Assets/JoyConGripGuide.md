# Joy-Con 握持模式設定指南

## 🎮 預設握持模式

在 Unity Inspector 的 `test.cs` 腳本中，選擇 **Grip Mode**：

### 1. **Horizontal_Right**（橫握 - 右手）⭐ 推薦
- **握法**：橫著拿，按鈕在右側
- **最常見的握法**
- 適合右撇子玩家

### 2. **Horizontal_Left**（橫握 - 左手）
- **握法**：橫著拿，按鈕在左側
- 適合左撇子玩家

### 3. **Vertical_Up**（直握 - 按鈕朝上）
- **握法**：直著拿，按鈕朝向天花板
- 類似傳統搖桿握法

### 4. **Vertical_Down**（直握 - 按鈕朝下）
- **握法**：直著拿，按鈕朝向地板
- 適合特殊遊戲玩法

### 5. **Custom**（自訂）
- 需要手動校正
- 按 **B 鍵**進行完整校正流程

---

## 🎯 開始遊戲前的設定步驟

### 步驟 1：選擇握持模式
1. 在 Unity Editor 中打開 `test.cs` 腳本的 Inspector
2. 找到 **Grip Mode** 下拉選單
3. 選擇你想要的握法（推薦 `Horizontal_Right`）

### 步驟 2：測試並微調
1. 按下 **Play** 開始遊戲
2. 如果光標飄移，按 **B 鍵**快速校正
3. 如果方向不對，按 **A 鍵**切換到下一個模式

---

## 🕹️ 遊戲中的按鍵功能

| 按鍵 | 功能 | 說明 |
|------|------|------|
| **A** | 切換握持模式 | 循環切換 5 種預設模式 |
| **B** | 快速校正 | 校正陀螺儀偏移（2秒） |
| **X** | 手動校正偏移 | 僅校正偏移值（3秒） |
| **Y** | 暫停/繼續 | 暫停遊戲 |

---

## 💡 常見問題

### Q: 光標一直飄移怎麼辦？
**A:** 按 **B 鍵**進行快速校正（只需 2 秒）

### Q: 方向完全相反怎麼辦？
**A:** 按 **A 鍵**切換到下一個握持模式

### Q: 想換個姿勢拿手把？
**A:** 按 **A 鍵**找到對應的握持模式，或選擇 `Custom` 模式後按 **B 鍵**

### Q: 校正時需要做什麼？
**A:** 
- 按 **B 鍵** → 保持靜止 2 秒 → 完成！
- 按 **X 鍵** → 保持靜止 3 秒 → 完成！

## 🔧 開發者備註

### 預設軸映射設定
```csharp
// Horizontal_Right（最常見）
axisMapping = new Vector2Int(2, 1); // X軸用Z, Y軸用Y
axisSign = new Vector2(1, 1);       // 都是正向

// Horizontal_Left
axisMapping = new Vector2Int(2, 1);
axisSign = new Vector2(-1, -1);     // 方向相反

// Vertical_Up
axisMapping = new Vector2Int(1, 2); // X軸用Y, Y軸用Z
axisSign = new Vector2(1, 1);

// Vertical_Down
axisMapping = new Vector2Int(1, 2);
axisSign = new Vector2(-1, -1);
```

### 優點
- ✅ 不依賴 orientation（穩定不飄移）
- ✅ 預設模式涵蓋 90% 使用情境
- ✅ 玩家可快速切換（按 A 鍵）
- ✅ 校正時間短（2-3 秒）

### 注意事項
- 預設模式基於標準握法測試
- 如果玩家習慣特殊握法，可使用 Custom 模式
- Custom 模式需要完整校正流程（約 6 秒）
